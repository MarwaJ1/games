# Projects 

In this repo there are multipal projets aiming to imporve coding skills and problem solving. 
Some of said projects are done and some aren't. 

## Radius Calculator 

You can find the radius calculator project in file "radius.py". By inputing the object (either a globe, cylinder or cone), volume and a height (no need to add one for a globe) it will return an aproximet radius for the object. 

## Tax counter 

You can find the swedish tax calvculator in the file "tax_calc.py". Using the function in said file add your income and you'll get an aproximet tax baised on the swedish taxing system. You'll also get a message either "Yikes!" or "Good for you!"

## Rock, Paper, Scissors 

You can find the classical Rock, paper, scissors game in the file "rps_game.py". Thsi game isnt yet finished. 

## Calculator 

You can find this simple calculator in the file "calc.py". You can use addition, subtraction, multiplication, disvison, squaring, to the power of. 

#### Addition 
Write a or A to be able to add x + y

#### Subtraction 
Write s or S to be able to sub x - y 

#### Multiplication 
Write m or M to be able to multiply x * y 

#### Divison 
Write d or D to be able to divide x/y 

#### To the power of 

Wite p or P to be able to square x, one input 

#### Square root 

Write sq to squre x, one input
